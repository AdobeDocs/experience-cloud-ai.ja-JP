# CX Coworker GatewayのReal-Time CDPツール {#rtcdp-mcp}

Real-Time CDP MCP製品ツールを使用して、MCP互換クライアントからオーディエンス、宛先、ソース、ID名前空間、アクティベーションの正常性を調べることができます。 これらのツールは、組織が有効になっていて、ユーザーアカウントに必要なReal-Time CDP権限が付与されている場合、統合[CX Coworker Gateway ゲートウェイ ](overview.md)を通じて利用できます。

>[!AVAILABILITY]
>
>Real-Time CDPの製品ツールはBetaに搭載されています。 アクセスは招待状によってのみ行われ、Adobe組織の有効化が必要です。 [CX Coworker Gateway Tools](access.md)へのアクセスを参照してください。

## 主な機能 {#mcp-capabilities}

Real-Time CDP ツールは、読み取り専用のモニタリングおよびトリアージ サーフェスを提供します。 以下を行うことができます。

* ライフサイクルの状態、生成元、ID名前空間などのオーディエンスを一覧表示して検査します。
* オーディエンスの評価を確認し、ジョブをエクスポートして、最近の失敗を特定します。
* 設定された宛先アカウント、宛先フロー、アクティベーションの実行履歴を調べます。
* ソースコネクタ、アカウント、フロー、取り込みの実行履歴を調べます。
* ID名前空間と結合ポリシーのリスト。

>[!IMPORTANT]
>
>現在のBetaのすべてのReal-Time CDP ツールは読み取り専用です。 オーディエンス、宛先、ソース、またはデータフローの作成、更新、アクティブ化、または削除はサポートされていません。

## 利用可能なツール {#mcp-tools}

| 面グラフ | ツール | 説明 |
| --- | --- | --- |
| オーディエンス | `search_audiences` | 名前、エンティティの種類、ライフサイクルの状態、ID名前空間、オリジンごとにオーディエンスをリスト化して検索できます。 |
| オーディエンス | `preview_audience_membership` | オーディエンスとして保存する前に、PQLまたはSDD セグメント式のサイズを見積もります。 |
| オーディエンス | `inspect_audience_evaluation_jobs` | セグメント評価ジョブレコードを取得して、オーディエンスの更新の問題を診断したり、最近の評価履歴を確認したりできます。 |
| オーディエンス | `inspect_audience_export_jobs` | オーディエンス書き出しのジョブレコードを取得して、書き出しが完了したか、失敗の詳細が表示されたかを確認します。 |
| 宛先 | `search_destination_connectors` | プラットフォームで使用可能な宛先コネクタタイプのリスト。 |
| 宛先 | `search_destination_accounts` | 認証済み宛先アカウントのリスト。 |
| 宛先 | `search_destination_input_connections` | 宛先フローのExperience Platform側の入力を取得します。 |
| 宛先 | `search_destination_output_connections` | ターゲットパス、ファイル形式、配信設定など、宛先フローの外部エンドポイントを取得します。 |
| 宛先 | `search_destination_flows` | 状態、マッピング、スケジュールなど、設定された宛先アクティベーションフローを一覧表示して調べます。 |
| 宛先とソース | `inspect_flow_runs` | ステータス、タイミング、レコード数、エラーの詳細など、ソースおよび宛先フローの実行履歴を取得します。 |
| ソース | `search_source_connectors` | プラットフォームで使用可能なソースコネクタタイプを一覧表示します。 |
| ソース | `search_source_accounts` | 認証されたソースアカウントのリスト。 |
| ソース | `search_source_input_connections` | ソースフローがアカウントから取得するものを取得します。 |
| ソース | `search_source_output_connections` | ソースフローのExperience Platform データセットの宛先を取得します。 |
| ソース | `search_source_flows` | 状態、マッピング、スケジュールなど、設定済みのソース取り込みパイプラインを一覧表示および検査します。 |
| ID | `search_identity_namespaces` | サンドボックスにID名前空間定義を一覧表示します。 |
| ID | `search_merge_policies` | リアルタイム顧客プロファイルの組み立て方法を制御する結合ポリシーレコードのリスト。 |

## プロンプト例 {#mcp-use-cases}

| 目標 | プロンプト例 |
| --- | --- |
| オーディエンスのリスト | 「`prod` サンドボックス内のオーディエンスを一覧表示します。」 |
| オーディエンスの調査 | 「オーディエンス ID `abc123`の詳細とライフサイクルの状態を表示する」 |
| 評価の問題を診断 | 「最新のオーディエンス評価ジョブとフラグの失敗を表示する。」 |

| オーディエンスサイズの推定 | 「保存する前に、このPQL式のサイズを見積もってください：`homeAddress.country = 'US'`」 |

## 権限 {#mcp-context}

Adobeの組織とサンドボックスのコンテキストは、ゲートウェイ接続レベルで一度確立され、すべてのツールファミリーに適用されます。そのため、Real-Time CDP ツールから組織やサンドボックスを切り替えることはできません。 セッションのコンテキストを設定するには、[ ツール呼び出しの製品コンテキスト ](install.md#mcp-connect-params)を参照してください。

照会するReal-Time CDP リソースを表示するには、ユーザーアカウントに権限が必要です。 ゲートウェイは製品の権限をバイパスしません。

## 既知の制限事項 {#mcp-limitations}

| 制限事項 | 説明 | 回避策 |
| --- | --- | --- |
| 読み取り専用サーフェス | Real-Time CDP ツールは、取得APIのみを公開します。 オーディエンス、宛先、ソース、またはデータフローを作成、更新、アクティベート、または削除することはできません。 | Real-Time CDP UIまたはExperience Platform APIを使用した書き込み操作。 |
| エンゲージメントや配信指標はない | ツールは、配信先のプラットフォームから、ダウンストリームの配信統計、エンゲージメント、コンバージョン指標を返しません。 | 宛先プラットフォームのレポート、Customer Journey Analytics ツール、Adobe Analytics ツールを使用して、エンゲージメントとコンバージョンデータを取得します。 |
| セグメントクエリは外部で作成する必要があります | `preview_audience_membership`には、有効なPQLまたはSDD式が必要です。 ツールはクエリを作成しません。 | セグメントビルダーまたはSegmentation Service APIで式を作成し、プロンプトに貼り付けます。 |
| 継続トークンによるページ化 | リストツールは、ページ分割された結果を返します。 非常に大きなサンドボックス全体で完全な列挙を行うには、連続トークンをチェーンする必要があります。 | 名前、状態、接続仕様、時間範囲などのフィルターを使用してクエリを絞り込みます。 |
| アクティベーション実行フィルタリングは時間ベースのみです | アクティベーション実行インスペクションでは、ステータスと完了タイムスタンプによるフィルタリングがサポートされますが、エラータイプや宛先プラットフォームによるフィルタリングはサポートされていません。 | 最初に`flowId`でフィルターを実行して、特定の宛先フローに実行をスコープで絞り込みます。 |
