# Data Engineering Agent

&lt;! – これはドラフトです – >

Data Engineering Agentは、Adobe Experience Platformのデータチーム向けに、Agent Orchestratorを利用した AI ベースのコパイロットです。 エージェントは、エンドツーエンドのデータライフサイクル（データモデリング、オンボーディング、SQL データ準備、ガバナンス、ライフサイクル管理）を通じて手間のかかる作業を処理するように設計されています。 Data Engineering Agentを使用すると、エンジニアとアーキテクトは、データ品質が高く、手作業が少なく、より迅速に作業を進めることができます。

Data Engineering Agentは、ワークフローを最適化するために利用できるスキルに整理されています。

| Data Engineering Agent skill | 説明 |
| --- | --- |
| データのオンボード | <strong> データオンボーディング </strong> では、S3、Marketo、Data Landing Zone （DLZ）などのバッチソースをサポートしています。 次のような機能があります。 <ul><li>S3、Data Landing Zone、Marketoなど、様々なデータソースへの接続。</li><li>ソースデータのプロファイル （例：個別の値、null、重複、基本的な品質指標）。</li><li>ソースフィールドを、標準 XDM フィールドグループとカスタムフィールドの両方に整列します。</li><li>データ取得データフローの提案と構築（例えば、S3 またはMarketoからRTCDP/CJA データセットへのデータの移動）。</li></ul> |
| 自動データ品質とセマンティック エンリッチメント | <ul><li>異常値の特定や有効/無効なレコード数のレポートなど、包括的なデータ品質評価を実行します。</li><li>では、タイプの正規化、通貨フォーマット、ID の重複排除など、データ品質の向上をお勧めします。</li><li>は、データサンプリングとインテリジェントな命名パターン認識に基づいて、スキーマにセマンティックエンリッチメントを適用します。</li><li>ユーザーに提案されたエンリッチメントの確認を依頼し、確定および承認済みのデータ変更を確認します。</li></ul> |
| Data Distiller | |
| データ収集 | |
| データの検証 | |

Data Engineering Agentを使用すると、次のことが可能です。

- **オンボーディングの高速化**：数週間にわたる手動スキーママッピングやパイプライン設定から、ガイドによる会話型の設定時間に至る時間を短縮します。
- **データ品質の向上**：ビルトインのデータ品質分析とセマンティックチェックにより、無効なレコードや実稼動インシデントが少なくなります。
- **ガバナンスとコンプライアンスの向上**：ガバナンスポリシーは、後になって考えるのではなく、デザイン時（ラベル、TTL、ID 処理）に付加されます。
- **生産性の高いデータチーム**：繰り返し行われる SQL/データフローの作業が自動化されるため、エンジニアはより価値の高い設計と最適化に専念できます。
- **より広範なセルフサービス**：エキスパートでない関係者でも、ガードレールを使用して、共通のデータ準備タスクを安全にセルフサービスで実行できます。
